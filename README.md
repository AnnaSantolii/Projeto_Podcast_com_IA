# Projeto_Podcast_com_IA
Projeto Dio - Podcast gerado com Inteligência Artificial

•	Resumo: 
Podcast criado com Inteligência Artificial, se tratando de Projeto voltado à conclusão do Curso UniversIA - Fundamentos de IA Generativa, disponibilizado pela Dio. Em parceira com o Santander Open Academy. 

•	Prompt de criação de nome do Podcast (via ChatGPT): 
Você é um roteirista de podcast e vamos criar um podcast de tecnologia focado em análise de dados e gostaria de uma ajuda para criar 5 sugestões de nomes criativos para esse podcast, com uma temática mais nerd. Use um trocadilho no nome.
O podcast vai falar sobre dicas, curiosidades e novidades sobre o mundo da análise de dados e o que está acontecendo no mercado.
{REGRAS}
Tanto o nome quanto o subtítulo deve ser enxuto.
O nome tenha algum trocadilho nerd com nomes de franquias ou animes conhecidos, como star wars, senhor dos aneis, nana, demon slayer, spy family etc.
O nome deve conter alguma palavra forte que remeta a análise de dados.
{REGRAS NEGATIVAS}
Foque na utilização de palavras em português (PT-BR).
Não quero que utilize nenhuma das palavras a seguir: sql, developer, python, R, programador.

•	Prompt de criação de imagem realista (via Midjourney):
A young Woman podcaster, redhead, average height, with black-framed glasses, slightly smiling, in a nerdy setting, gamer style, some low light, a professional microphone, dynamic cinematic lighting, ultra-high resolution, realistic true-color filter, 16K photo processing, hyper-realistic, ultra-detailed, —ar 1:1 —v 5.2

•	Prompt de criação de roteiro do Podcast (via ChatGPT): 
Você é um roteirista de podcast e vamos criar um roteiro de um podcast de tecnologia focado em análise de dados, cujo nome é “Data Wars: A Guerra dos Dados”, com público alvo de iniciantes em análise de dados.
O formato do roteiro deve ser:
[INTRODUÇÃO]
[CURIOSIDADE 01]
[CURIOSIDADE 02]
[NOVIDADE]
[FINALIZAÇÃO]
{REGRAS}
No bloco [INTRODUÇÃO] substitua por uma introdução igual às introduções dos vídeos do canal “Ei, Nerd”, como se fossem escritos pelo Peter Jordan.
No bloco [CURIOSIDADE 01] substitua por uma curiosidade de análise de dados
No bloco [CURIOSIDADE 02] substitua sobre uma ferramenta de análise de dados, podem ser softwares, linguagem de dados etc.
No bloco [NOVIDADE] aborde sobre algum tema recente que tenha relação direta com a análise de dados.
No bloco [FINALIZAÇÃO] substitua por uma despedida divertida com o final “Eu sou a Anna e esse foi o DATA WARS dessa semana”
Use termos de fácil explicação e entendimento
O podcast vai ser apresentado somente por uma pessoa chamada Anna
O podcast deve ser curto
{REGRAS NEGATIVAS}
Não use muitos termos técnicos
Não ultrapasse 5 minutos de duração

•	Demais plataformas/softwares utilizados: 
Notion Template para organização de ideias; 
Capcut para edição de áudio.
